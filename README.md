# TinyBox - 小巧工具箱

一款 Windows 桌面轻量工具应用，集成便签、待办计时、剪贴板历史、文件暂存、网易云音乐播放等功能。

UI 采用灵动岛风格（类 iPhone Dynamic Island），科技感暗色主题，常驻屏幕侧边栏。

## 特性

- **便签** - 快速记录，支持颜色选择、置顶、搜索
- **待办 + 番茄钟** - 任务管理 + 25分钟工作/5分钟休息计时
- **剪贴板历史** - 自动监听剪贴板变化，支持收藏、搜索、最多100条
- **文件暂存** - 拖拽文件临时保存，随时拖出使用
- **网易云音乐** - QR码登录、歌单浏览、音频播放、实时歌词同步
- **发现页** - 推荐歌单、每日推荐歌曲、热搜关键词
- **全局快捷键** - Alt+Space 显示/隐藏、Ctrl+Shift+V/M/N/L 等
- **设置面板** - 窗口置顶、剪贴板监听、番茄钟时长、快捷键重绑定
- **迷你播放器** - 侧边栏收起时显示当前歌曲封面

## 技术栈

| 组件 | 技术 |
|------|------|
| 桌面框架 | Tauri 2.0 (Rust + WebView2) |
| 前端 | Svelte 5 + TypeScript |
| 样式 | Tailwind CSS 4 |
| 数据库 | SQLite (tauri-plugin-sql) |
| 音乐 API | ncm-api-rs |
| 音频播放 | rodio |
| 安装包 | NSIS (~9.6MB) |

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run tauri dev

# 构建
npm run tauri build
```

### 环境要求

- Node.js 22+
- Rust (stable)
- Windows 10/11 (WebView2)

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Alt+Space | 显示/隐藏侧边栏 |
| Ctrl+Shift+V | 打开剪贴板历史 |
| Ctrl+Shift+N | 新建便签 |
| Ctrl+Shift+M | 播放/暂停音乐 |
| Ctrl+Shift+L | 显示歌词 |

所有快捷键可在设置面板中自定义。
